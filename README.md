# Academic Personal Homepage Template

这是一个接近经典学术主页风格的 GitHub Pages 模板。

## 文件说明

```text
index.html        页面内容
style.css         页面样式
script.js         自动年份脚本
assets/           图片、头像、论文图、实习机构图
```

## 你主要需要修改的地方

1. `index.html` 里的 `Your Name`、邮箱、GitHub 链接、学校/单位信息。
2. 把你的头像上传到 `assets/profile.jpg`，然后把 HTML 中的头像路径改为 `assets/profile.jpg`。
3. 把论文图上传到 `assets/paper-1.jpg`、`assets/paper-2.jpg` 等，再修改对应 `img src`。
4. 把实习机构 logo 或实习项目图上传到 `assets/company-1.png`、`assets/intern-project-1.jpg` 等，再修改对应 `img src`。
5. 修改 `News`、`Selected Publications`、`Internship Experience`、`Honors & Awards`、`Academic Services` 的内容。

## GitHub Pages 使用方法

把所有文件放到 `你的用户名.github.io` 仓库根目录，确保 `index.html` 在根目录。然后在仓库 `Settings -> Pages` 中选择 `Deploy from a branch`，分支选择 `main`，目录选择 `/(root)`。
